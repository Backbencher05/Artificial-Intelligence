Session 4
Career Paths in AI

Before learning anything else, we need to answer:

"What exactly am I becoming?"

Because AI is NOT one job.

Many people say:

"I want to get into AI."

But they don't know which path.

That is like saying:

"I want to get into medicine."

Doctor?
Surgeon?
Radiologist?
Cardiologist?

AI is similar.

The AI Career Landscape
Artificial Intelligence
│
├── Data Analyst
├── Data Scientist
├── Machine Learning Engineer
├── Deep Learning Engineer
├── AI Research Scientist
├── Generative AI Engineer
├── LLM Engineer
├── AI Backend Engineer
├── MLOps Engineer
├── AI Solutions Architect
├── AI Automation Engineer
├── AI Product Manager
└── AI Entrepreneur

Let's understand each.

1. Data Analyst
What They Do

Answer:

"What happened?"

They:

Analyze business data
Create dashboards
Find trends
Generate reports
Skills
SQL
Excel
Power BI
Tableau
Basic Python
Example

"Why did sales drop last month?"

Should YOU choose this?

❌ No.

Reason:

Your backend experience is much more valuable.

2. Data Scientist
What They Do

Answer:

"Why did this happen?"

and

"What will happen?"

They:

Build prediction models
Analyze data
Conduct experiments
Skills
Statistics
Machine Learning
Python
Scikit-learn
Example

Predict:

Customer churn
Loan default
Fraud detection
Should YOU choose this?

⚠️ Learn foundations.

But not your primary path.
