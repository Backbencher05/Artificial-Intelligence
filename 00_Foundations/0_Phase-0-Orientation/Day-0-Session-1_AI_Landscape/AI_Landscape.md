# First Question
What is Intelligence?
-------------------------------
Let's start with humans.

Imagine a child.

The child:

sees objects
hears sounds
learns patterns
remembers experiences
makes decisions

Example:

A child touches fire.

Result:

feels pain
remembers it
avoids touching it again

The child learned from experience.

This ability to learn from experience is one form of intelligence.

# What is Artificial Intelligence?

Artificial Intelligence is:

Building machines that can perform tasks that normally require human intelligence.

Notice something important:

We are NOT trying to create humans.

We are trying to replicate specific intelligent behaviors.

Examples:

Human Ability	          AI Equivalent
- Recognize faces: 	    Face recognition
- Understand language: 	ChatGPT
- Drive a vehicle:  	Self-driving cars
- Play chess:       	Chess AI
- Recommend products:	Netflix recommendations
- Translate language: 	Translation systems

The Biggest Misconception

Many people think AI means:

The machine thinks.

Not necessarily.

In many cases:

The machine simply learned patterns from massive amounts of data.

Example:

When ChatGPT answers:

It isn't "thinking" like a human.

It is predicting what token (word piece) should come next based on patterns it learned.

## Evolution of AI
---------------------------------
To understand today's hype, we need history.

# - Era 1: Rule-Based AI
--------------------------
Old systems worked like this:

IF temperature > 100
THEN alert

Everything was manually programmed.

Problem:

They couldn't learn.

# - Era 2: Machine Learning
-------------------------------------
Humans stopped writing every rule.

Instead:

We gave machines data.

Example:

Thousands of spam emails.

The system learned:

What makes an email spam?

This was revolutionary.

# - Era 3: Deep Learning
---------------------------------------
More powerful neural networks.

Huge amounts of data.

Huge computing power.

This enabled:

image recognition
speech recognition
translation

# - Era 4: Generative AI

The current era.

Now machines can:

write
draw
code
summarize
generate video
generate audio

This is what created the AI explosion.




# Why AI Exploded Recently

Three things happened simultaneously:

1. Massive Data

Internet data became enormous.

Websites.
Books.
Code.
Articles.
Videos.

Models could learn from all of it.

2. Massive Computing Power

GPUs became extremely powerful.

Training that once took years became feasible.

A lot of this acceleration came from advances driven by companies like NVIDIA.

3. Transformer Architecture

In 2017, researchers introduced a breakthrough architecture called:

Attention Is All You Need

This led to modern LLMs.

We'll study this later in depth.

For now:

Know that this was a turning point in AI history.
