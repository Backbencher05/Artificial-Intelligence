# Engineering Checkpoint — Day 001

### 1. What new mental model did I build today?

Today, I realized that learning is fundamentally the process of observing experiences, identifying patterns, and using those patterns to make predictions about new situations. Machine Learning follows the same principle—it enables computers to learn patterns from data instead of relying solely on explicitly programmed rules.

---

### 2. What misconception did I correct today?

Earlier, I thought Machine Learning was simply another branch of programming with more advanced algorithms. Today, I understood that it represents a different approach to solving problems. Instead of manually writing every rule, we provide examples and allow the system to discover the underlying patterns on its own.

---

### 3. What question is still confusing me?

If a machine learns from examples rather than explicit rules, what exactly does it store internally after training? Does it memorize important examples, create mathematical relationships, or build something entirely different? I'm curious to understand what a trained model actually contains.

---

### 4. If I had to teach today's lesson in 2 minutes, what would I say?

Machine Learning begins with a simple idea that humans naturally follow: we learn from experience. By observing many examples, we recognize patterns and use those patterns to predict future outcomes. Instead of programming every possible rule, we let the machine learn these patterns from data. In essence, Machine Learning is a way of solving problems where writing explicit rules is difficult, but learning from examples is possible.
