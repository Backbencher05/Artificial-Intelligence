## Phase 1 – Python for AI & Data Foundations
Goal of this Phase

By the end of this phase, you should be able to:

- Work comfortably with AI data structures
- Manipulate datasets efficiently
- Understand how data flows through ML systems
- Understand the difference between Python lists, NumPy arrays, and tensors
- Visualize data
- Perform Exploratory Data Analysis (EDA)

Be fully prepared for Machine Learning
